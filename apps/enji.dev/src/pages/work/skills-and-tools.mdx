## Front-End Developer

I learned MERN stack within my 6-months internship , plus Angular , where the details include:

### JavaScript

At the very first, I learned JavaScript which is the core of building web applications, and the core of 
MERN stack , adding such technologies to JS helped enhancing the functionality, user experience 
and ease of code.

### React

React.js is my go-to front-end library due to its ease of implementation, flexible structure, 
and seamless UI development. It leverages JSX as a declarative markup syntax, 
making component-based development more intuitive and efficient.


### Node.js
Building scalable back-end services? Node.js makes it a breeze. Been working with it, and its speed and efficiency never fail to impress!

### Express.js
Minimal yet powerful—Express.js helps me structure APIs like a pro. I’ve been using it to create seamless back-end solutions with ease!

### MongoDB
Data storage made flexible! MongoDB lets me handle structured and unstructured data effortlessly. I’ve been leveraging it for dynamic applications, and it’s been a game-changer.

### Tailwind CSS
Utility-first styling is my thing, and Tailwind CSS makes it effortless. Been using it to create sleek, responsive designs with minimal effort!

### Bootstrap
Rapid prototyping? Bootstrap has my back. I’ve been using it to craft clean and responsive layouts in no time!

### CSS
Styling from scratch? Love it! CSS lets me bring creative ideas to life, and I enjoy making pixel-perfect designs.

### TypeScript
Stronger types, fewer bugs—that’s why I rely on TypeScript. Been using it to write cleaner, more maintainable code!

### Firebase
Real-time database, authentication, and hosting in one? Firebase keeps things simple, and I love how smooth it makes app development!

### Git & GitHub
Version control is essential, and Git & GitHub keep my projects organized. Been using them to collaborate and manage code efficiently!
