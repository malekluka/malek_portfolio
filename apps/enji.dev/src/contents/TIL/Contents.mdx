    ## Angular Signals – A Simpler Way to Handle Reactive State
    <TIL.ItemTags>
      <TIL.ItemTag>Angular</TIL.ItemTag>
      <TIL.ItemTag>Signals</TIL.ItemTag>
      <TIL.ItemTag>State Management</TIL.ItemTag>
    </TIL.ItemTags>

    Angular Signals offer a new, intuitive way to manage and react to state changes without the complexity of RxJS.
    Signals are synchronous, easier to debug, and eliminate the need for manual subscriptions.

    ````ts
    import { signal } from '@angular/core';

    export class CounterComponent {
      count = signal(0);

      increment() {
        this.count.set(this.count() + 1);
      }

      decrement() {
        this.count.set(this.count() - 1);
      }
    }
    ````

    [Official Docs](https://angular.dev/guide/signals)

    ## React useEffect Hook – Side Effects Made Easy
    <TIL.ItemTags>
      <TIL.ItemTag>React</TIL.ItemTag>
      <TIL.ItemTag>useEffect</TIL.ItemTag>
      <TIL.ItemTag>Hooks</TIL.ItemTag>
    </TIL.ItemTags>

    `useEffect` lets you perform side effects in functional components like data fetching, subscriptions, or manually changing the DOM.

    ```js
    import { useEffect, useState } from 'react';

    function Example() {
      const [count, setCount] = useState(0);

      useEffect(() => {
        document.title = `You clicked ${count} times`;
      }, [count]);

      return <button onClick={() => setCount(count + 1)}>Click me</button>;
    }
    ```

    [Official Docs](https://react.dev/reference/react/useEffect)
